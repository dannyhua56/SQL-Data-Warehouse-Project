# SQL-Data-Warehouse-Project
Building a modern data warehouse with SQL Server, including ETL processes, data modeling and analytics.

Hello this is my Data Warehouse Analystics Project repository!
This project will demostrate a very comprehensive usage for data warehousing and analytic solution. Start 
to finish processes will consist of bronze -> silver -> gold.

| Layer        | Description |
|-------------|------------|
| Bronze Layer | Holds raw data ingested from CSV files into SQL with minimal transformation |
| Silver Layer | Cleans, filters, and standardizes data to ensure consistency and usability |
| Gold Layer   | Final curated dataset modeled into a star schema for reporting and analytics |
