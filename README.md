# SQL-Data-Warehouse-Project
Building a modern data warehouse with SQL Server, including ETL processes, data modeling and analytics.

Hello this is my Data Warehouse Analystics Project repository!
This project will demostrate a very comprehensive usage for data warehousing and analytic solution. Start 
to finish processes will consist of bronze -> silver -> gold.

**BRONZE LAYER:** Holds the raw dataset in CSV files that must be imported into SQL.
**SILVER LAYER:** Main objective is to clean, filter and standardize the entire dataset in order to prepare
for analysis (my favorite part)
**GOLD LAYER:** Dataset is all filters, clean on joined together! data is modeled into a star schema 
required for reporting and analytics.
